# E-commerce-Sales-Customer-Intelligence-System


##  Overview
This is an end-to-end data analytics project that analyzes sales data to extract meaningful business insights. The project covers data extraction, cleaning, analysis, and visualization using multiple tools.

##  Tools & Technologies
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI
- Streamlit

---

##  Key Features
- KPI metrics (Revenue, Orders, Customers)
- Sales trend analysis
- Top-performing products
- Customer segmentation insights
- Country-wise revenue analysis
- Interactive filters (Month, Country)


---

## 💡 Key Insights
- A small number of products contribute to a large portion of revenue
- Sales show seasonal trends across months
- High-value customers drive the majority of business revenue
- Revenue is concentrated in specific regions
